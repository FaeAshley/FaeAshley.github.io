Weekly Time Tracking App

A weekly time tracker web app built with vanilla JavaScript, HTML, and CSS, designed to log multiple work sessions per day with clock-in/out times, task types, and automatic hour calculations. Features include dynamic time slot creation, dropdown task labeling (e.g., Work, Break, Lunch), weekly navigation, total hour summaries, and localStorage persistence for data retention without a backend.
